# Community CPU Pulse

This sample behaves like a third-party local widget. It requests only the `cpu` system metric, executes its JavaScript provider outside Explorer, and renders through the native JSON layout DSL.

Install `com.pfc.community-cpu-pulse-1.0.0.twidget` from **Settings > Developer > Install .twidget**. When enabling it, approve the displayed `systemMetrics: cpu` permission.
